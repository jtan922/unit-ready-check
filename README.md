# Pre-Show / Pre-Move-In Inspection Sheet

This is a separate unit inspection app from the weekly Vale property inspection template. The sheet layout follows the `PRE-SHOW/OCCUPANCY INSPECTION` PDF, with the saved-unit controls kept outside the printable form.

## Use

- Open `index.html` in a browser.
- Start a new unit from the Saved Units panel.
- Fill in the PDF-style inspection sheet over one visit or several days.
- The sheet auto-saves in the browser, and the Save button forces a save.
- Reopen the same browser and select the unit from Saved Units to continue.
- Use Backup All to export saved inspections as JSON.
- Use Import to restore a JSON backup.
- Use Print / Save PDF when the unit is ready to file.

## Notes

The saved units live in the browser storage on the device you use. For long-term records or moving to another device, use Backup All.
